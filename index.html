AboutX

Nurikabe is a hobby project and is not complete yet. My goal is to add 100 puzzles each in the easy (5x5), medium (7x7) and difficult (9x9) category.

If you find any issues with the solutions or want to suggest any changes or updates to the code, please contact me at aditya.shevade[at]gmail.com.

RulesX

Basic rules of Nurikabe are pretty simple and can be summed up in the following key points:

  • Each island contains exactly one clue.
  • The number of squares in each island equals the value of the clue.
  • All islands are isolated from each other horizontally and vertically.
  • There are no wall areas of 2x2 or larger.
  • When completed, all walls form a continuous path.
